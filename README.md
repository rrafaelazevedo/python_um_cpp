# python_um_cpp

Repositório com materiais introdutórios de **Python** e **modelagem matemática elementar**, com foco em conteúdos didáticos e exemplos práticos em notebooks.

## Sobre o projeto

Este repositório reúne conteúdos de apoio para estudo de:

- conceitos básicos de Python;
- fundamentos de programação científica;
- noções de modelagem matemática;
- uso de bibliotecas comuns no ecossistema Python.

## Estrutura do repositório

Na raiz do projeto você encontra materiais em PDF e pastas temáticas, incluindo:

- `python_basico/`
- `numpy/`
- `pandas/`
- `modelagens/`
- `modelagens_computacionais/`

Além disso, há apostilas e resumos em PDF, como:

- `variaveis.pdf`
- `funcoes_def.pdf`
- `funcoes_lambda.pdf`
- `objetos.pdf`
- `glossario.pdf`
- entre outros.

## Como usar

1. Navegue pelas pastas de acordo com o tema desejado.
2. Abra os notebooks (`.ipynb`) no Jupyter Notebook ou no Google Colab.
3. Use os PDFs como apoio teórico para os exercícios e exemplos.

## Requisitos (opcional)

Para executar os notebooks localmente, recomenda-se:

- Python 3.10+
- Jupyter Notebook ou JupyterLab
- Bibliotecas:
  - `numpy`
  - `pandas`

Exemplo de instalação:

```bash
pip install notebook numpy pandas
```

## Objetivo

Servir como base de estudos para quem está começando em Python e deseja aplicar programação em problemas de matemática básica e modelagem.
